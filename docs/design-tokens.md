# Design Tokens — Tesla-Inspired

## Colors
- Electric Blue: #3E6AE1 (primary CTA)
- Pure White: #FFFFFF (background)
- Carbon Dark: #171A20 (headings, nav)
- Graphite: #393C41 (body text)
- Pewter: #5C5E62 (tertiary text)
- Silver Fog: #8E8E8E (placeholder)
- Light Ash: #F4F4F4 (alt surface)
- Cloud Gray: #EEEEEE (borders)
- Pale Silver: #D0D1D2 (subtle borders)

## Typography
- Font: Inter (closest to Universal Sans available via Google Fonts)
- Hero: 40px / 500 / 1.2
- Product: 17px / 500 / 1.18
- Nav: 14px / 500 / 1.2
- Body: 14px / 400 / 1.43
- Button: 14px / 500 / 1.2

## Spacing: 8px base
## Border Radius: 4px (buttons), 12px (cards)
## Transitions: 0.33s cubic-bezier(0.5, 0, 0, 0.75)
## Shadows: none (flat design)
## Depth: opacity-based only
