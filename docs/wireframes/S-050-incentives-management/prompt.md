## 設計意図メモ

- 未調整キューは最上段に固定。共同開催の手動調整漏れを月次確定前に必ず潰す導線
- 二次店別行を展開すると対象契約リスト。インセンティブの根拠を契約レベルまで追跡可能
- 「下書き / 確定 / 取消 / 負調整」の 4 状態を別カラムで集計。F-046 / F-043（キャンセル期限後の負調整）に対応
- 月次確定済の行はロック表示（ワイヤーでは未確定で例示）
