## 設計意図メモ

- マスタ系一覧の標準形。検索 + フィルタ + テーブル + ページネーション
- 契約形態列で固定費/成果報酬/混合が一目で判別。金額と率の表示位置を統一
- 無効レコードは灰色化して残し、過去履歴を保持
