## 設計意図メモ

- 卸業者本部から見る一覧。自社案件と二次店案件を統合表示
- 二次店スコープ「アポ獲得まで」の場合は編集ボタンを灰色化 → 閲覧のみ
- 提案金額 + 粗利見込みを並列表示。クロージング前後の判断材料
- 集計バッジで「契約見込み / 失注 / 未着手」のヘルスチェック
