## 設計意図メモ

- 全テナント横断の監査ログ。運営者がインシデント調査に使う
- 個人情報は常時マスク固定（運営者でも見えない）。注記で明示
- テナント列を最前面に。「どのテナントの誰の操作か」を即把握
- 操作種別「ログイン (失敗 N 回)」も明示。セキュリティ調査に直結
