## 設計意図メモ

- 「上書きせず新規挿入」のルール（F-012）を価格履歴 UI で明示。タイムラインで全レコードを保持
- 適用中レコードは太枠 + バッジ「適用中」。視覚的に「これが今の値段」
- 「過去契約には反映されない」を注記カードで強調。契約明細スナップショット原則の周知
- 新価格追加は履歴に追記される動作。ユーザーが意図を持って操作することを徹底
