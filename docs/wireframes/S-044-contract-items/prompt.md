## 設計意図メモ

- 契約明細は契約ごとに完結。商品マスタとは独立した snapshot を持つことを SnapshotChip で常時明示
- 商品ピッカーは契約日時点で有効な商品のみ表示。F-041 のバリデーション
- 集計カードに「粗利計算は別画面」と明示。S-045 への自然な遷移を案内
- 明細 0 件のときは粗利計算ボタンが非活性（ワイヤーでは 4 件で例示）
