## 設計意図メモ

- F-036 の 24 時間ルールに基づき、経過時間列を必ず表示。warning しきい値（22h, 24h+）で視覚強調
- 配信プレビューで「二次店に何が送られるか」を事前確認可能。社内メモが漏れない保証
- 2 ペイン構造で連続処理に最適化（S-035 と同様）
- スマホでは右ペイン → 別画面遷移、確認ボタンを画面下に固定
