## 設計意図メモ

- 月単位の管理が業務サイクル。月切替を最上段に固定
- ステータスタブで「下書き / 受付中 / 受付終了 / 体制決定 / 中止」のフェーズ別ビューを切替
- 一括公開は下書きから受付中への遷移。複数件選択時に有効
- 希望提出数 (4 / 12 社) は「対象二次店中何社提出済み」の進捗バー的役割
