## 設計意図メモ

- 二次店向けイベント詳細。場所提供元の契約条件は表示しない（注記で明示）
- スコープ上書きが入っているイベントは大きく明示。「今回だけ範囲が広がる/狭まる」を理解してもらう
- 共同開催時の連絡先は業務用なのでマスク無し（業務通信を妨げない）
- 報告タブを最初に出して、現場担当が即座に報告できる導線を確保（S-063 へリンク）
