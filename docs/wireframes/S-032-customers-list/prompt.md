## 設計意図メモ

- 個人情報マスキングを徹底。氏名すらマスク表現（"山○ 太○"）でデモ画面の漏えいを避ける
- 鍵アイコンは権限あり時のみ活性。誰でもクリックできる位置に置きつつ操作は監査ログ
- 検索は氏名 + 電話下 4 桁 + 住所部分。日常的に「電話の下 4 桁で探す」運用が多いため
- 登録元列で「自社 / どの二次店 / どのイベント」を明示。後追い分析の起点
