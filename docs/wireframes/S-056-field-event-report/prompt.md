## 設計意図メモ

- 報告タブ 3 つ（開始 / 終了 / 成果）を並列。それぞれ独立に提出可能（提出済はチェックマーク）
- 成果報告は数値入力が中心。スマホで数字キーボードが立ち上がるよう型を numeric に
- バリデーション「有効 + 無効 ≤ アポ取得数」を error.png で例示
- 写真添付は最大 5 枚。Phase 1 で対応
- 開始未報告でも終了・成果報告は提出可能（業務上、開始打刻忘れがある）
