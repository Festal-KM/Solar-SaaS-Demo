## 設計意図メモ

- 新規作成と詳細編集を 1 画面で扱う（モード切替）。共通フィールドが多いため
- 招待状態カードで「招待メールがいつ送られて、受諾されているか」を即確認可能に
- プラン変更履歴は監査用途。誰がいつ何を変えたか追跡できる
- メール重複エラーはモーダル明示。既存テナントへの導線も提供して運営者が判断しやすく
