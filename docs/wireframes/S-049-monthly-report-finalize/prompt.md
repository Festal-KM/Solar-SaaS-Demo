## 設計意図メモ

- 確定アクションがこの画面の核。確定すると当月の編集がロックされる重みを「チェックリスト + 非活性ボタン」で UI 上にも反映
- 16 指標の自動集計を 4x4 グリッドで一覧化。月次報告書に転記する数値を即可視化
- 二次店コメントは「成果 / 課題 / 翌月施策」の 3 段組で型化
- 共同開催インセンティブ未調整がある場合は確定不可 → 警告 + S-045 への直接リンク
