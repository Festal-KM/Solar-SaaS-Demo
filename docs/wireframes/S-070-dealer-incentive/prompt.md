## 設計意図メモ

- 二次店にとって最重要画面。インセンティブの根拠を契約レベルまで追跡可能
- 支払予定額 (確定 - 負調整) を最上段で強調。月末の集計値として最も気にする数字
- 共同調整待ち（下書き）は warning。卸業者へのフォローアップ判断材料
- キャンセル期限後の負調整は destructive。「過去のインセンティブが取り消される」ことを明示
- 仕入値・施工費は完全非表示。F-051 / docs/01 §9 の情報開示制約
