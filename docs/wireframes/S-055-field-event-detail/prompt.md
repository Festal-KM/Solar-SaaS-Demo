## 設計意図メモ

- 現場到着時に最初に見る画面。地図リンクと開始報告ボタンを大きく配置
- 他スタッフの連絡先は電話 / メールアイコンで即連絡可能。共同開催の他社スタッフへの連携も同様
- アクションボタン群はカードの最下段に固定。スマホで親指の届く範囲
- 開始 / 終了 / 成果 報告のボタンを分離。報告タイミングが業務上明確に分かれているため
