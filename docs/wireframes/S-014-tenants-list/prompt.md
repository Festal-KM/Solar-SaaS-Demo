## 設計意図メモ

- 一覧画面の基本形。検索/フィルタ → テーブル → ページネーションを画面上から下へ自然に配置
- ステータス「休止」は淡グレーで薄く表示し視覚的に区別
- モバイルではテーブルではなくカードリストで指タップ前提に最適化
- バルク操作は今フェーズでは未定義のためチェックボックスのみ用意（将来拡張用）
