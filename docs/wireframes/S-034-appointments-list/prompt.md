## 設計意図メモ

- 訪問 24h 以内 + マエカク未対応はもっとも緊急。赤帯で視覚的に最優先化（destructive）
- 「コール部隊向けキュー」タブで S-035 への入口を兼ねる
- アポ取得者と組織を分離。同じ顧客でも誰がアポを取ったかで責任が分かれる
- フィルタは「期間 + ステータス + 担当 + 組織」の 4 軸。営業マネージャー視点
