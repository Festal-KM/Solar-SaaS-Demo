## 設計意図メモ

- 二次店別ビューは「どこが未提出か」を最優先で可視化。未提出は destructive 強調 + リマインドボタンを直置き
- 行展開で具体的な希望明細（候補日 × 店舗 × 人数）を確認できる。ドリルダウン UI
- 期限切れは灰色化。リマインドは引き続き可能（業務都合で受付延長することがある）
- 一括リマインドはバッジ件数表示で対象数を明示
