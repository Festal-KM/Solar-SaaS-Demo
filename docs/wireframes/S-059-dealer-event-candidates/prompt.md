## 設計意図メモ

- 二次店には「他社の希望状況」「配属状況」は一切見せない。F-020 の業務要件
- 表示情報は「日付 / 店舗 / エリア / 想定来場 / 回答期限」のみ。場所提供元の契約条件は非表示
- カードレイアウトで一覧性確保。テーブルより視認性が高い
- 提出済みカードは淡表示で区別。「希望を編集」で再提出可能（期限内のみ）
