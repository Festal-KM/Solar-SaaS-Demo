## 設計意図メモ

- チェック式 + 展開式の UI で「選んだ候補にだけ詳細入力」。未選択候補は折りたたまれて視認性確保
- 優先順位 / 対応可能日 / 対応可能人数 / コメントは任意。最小限の入力で提出可能
- サマリで「何件選んだか」を最後に確認できる導線
- 1 件以上必須のバリデーション。F-021 受入基準
- 提出済みでも期限内であれば再編集可能（S-059 から戻ってきて編集 → 再提出）
