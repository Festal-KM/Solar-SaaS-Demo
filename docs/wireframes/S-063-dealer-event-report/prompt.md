## 設計意図メモ

- S-056（自社現場）とほぼ同等の構造。フォームを共通化しつつロールで配信先を分岐
- 二次店のイベント報告は卸業者本部に通知される。F-028 / F-029 / F-030
- スマホ最適化（タブ切替 + 縦並び）。dealer_staff は下部タブバー、dealer_admin はサイドバーで遷移
- 開始 / 終了 / 成果 の 3 タブを並列。報告タイミングごとに独立提出
