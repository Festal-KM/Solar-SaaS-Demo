## 設計意図メモ

- マスタ詳細の標準形。基本情報 / 契約条件 / 備考 / 履歴の 4 セクション分離
- 契約形態に応じて固定費 or 成果報酬率の入力欄を活性切替（ワイヤーでは固定費活性、率非活性で例示）
- 変更履歴は監査ログとは別の「マスタ自身の変更履歴」。タイムライン UI で時系列に
