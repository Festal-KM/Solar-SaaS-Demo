## 設計意図メモ

- 月次サマリで「集計値の動き」を最初に見せる。詳細テーブルは具体個別
- 開始/終了/成果の 3 つを列で並べ、報告漏れを一目で発見
- 報告者は自社と二次店の両方を「/」区切りで表示。共同開催の責任所在を可視化
- 当日進行中（開始済み + 終了未）の行は太枠 highlight
