## 設計意図メモ

- 二次店担当（営業マン）はスマホ完結が大前提。当日イベントを最上段で強調
- 「クイック報告へ」「アポ顧客登録」を最大の CTA。1 タップで業務開始
- 自分が登録したアポのみ表示。担当責任の範囲を明確化
- 下部タブバーで主要動線を 5 つに圧縮（配属イベント / アポ / 顧客 / 通知 / その他）
