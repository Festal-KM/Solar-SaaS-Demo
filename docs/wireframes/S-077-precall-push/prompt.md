## 設計意図メモ

- プッシュ通知タップ → 単体表示。一覧 (S-066) を経由せず即確認できる導線
- 注意事項は最大の情報資産。フォントを大きめにし、強調表示
- 「確認しました」ボタンを最大の CTA。タップ → 状態更新 → S-066（または S-065）へ遷移
- コール部隊の社内メモは「非表示」と明示。情報開示境界の透明性を保つ
