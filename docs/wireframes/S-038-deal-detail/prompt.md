## 設計意図メモ

- 顧客 + アポを上段カードで再掲。商談画面単体で完結（前画面に戻らず判断可能）
- ステッパーで業務フェーズを可視化。横一列の順序が業務感覚と一致
- スコープが「アポ獲得まで」の二次店ロールは編集不可（このワイヤーは卸業者ビュー）
- 「契約に進む」ボタンは契約予定時のみ強調活性。S-040 新規契約に事前入力で遷移
