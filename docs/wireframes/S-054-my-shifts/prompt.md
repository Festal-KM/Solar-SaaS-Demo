## 設計意図メモ

- 自分のシフトのみ表示。プライバシー + 集中（他人のシフトに惑わされない）
- 今日のシフトは太枠 + 「今日」バッジで最強調。最初に視線が行く
- カード単位で 1 タップ → イベント詳細 (S-055) へ
- カスタム期間切替は将来拡張用（標準は今日 / 今週 / 今月）
