## 設計意図メモ

- 運営者は「何社いて、何社払ってない、何社放置されてる」を一目で見たい。KPI を最上段に
- 監査ログプレビューは PII マスク表示固定。運営者には個人情報を見せない設計（F-055）
- 「直近の監査ログ」と「最近作成テナント」を並べることで運営活動の動線が見える
- 空状態はチュートリアル化。最初の操作（テナント作成）を明確に CTA 化
