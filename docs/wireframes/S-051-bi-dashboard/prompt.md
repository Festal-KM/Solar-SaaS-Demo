## 設計意図メモ

- ドリルダウン UI を意識。BI 画面から各一覧画面（契約 / 月次 / イベント）へ遷移可能
- グラフは色なしで線種・濃度・棒長さで表現。色覚多様性に配慮
- 二次店ロールには非表示（卸業者本部のみ）
- CSV エクスポートで Phase 2 以降の外部 BI 連携を可能に
