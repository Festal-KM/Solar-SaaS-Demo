## 設計意図メモ

- 強度メーターは入力中に動的更新される想定。ワイヤーでは「強度: 中」を例示しユーザーにフィードバックがあることを示す
- リンク 1 回限り + 30 分有効をフォーム下に常時表示。error 画面で具体的な救済導線を示す
- 確認入力との不一致時はインラインエラーで示す（error.png ではリンク切れケースを優先）
