## desktop.png プロンプト

```
Layout: 1440x900 pixels.

Top bar (64px): 左 "Solar SaaS"、右にアバター `[山]` + リンク "ログアウト"。

中央コンテンツ (最大幅 900px):

### Section 1: ヘッダー
- 見出し "提携卸業者を選択"
- グレー "業務を行う卸業者を選択してください。後でヘッダー右上から切り替えられます。"

### Section 2: 卸業者カードリスト（3 カラム × 2 行 = 6 カード）
- カードレイアウト（各 280x180 px）、太枠四角:
  - 上部: 卸業者名（太字、大きめ）例: "株式会社サンライズソーラー"
  - 中部: ステータスバッジ `「有効」`（または淡グレーで `「停止中」`）
  - 中部: グレー "業務スコープ: 商談・クロージングまで"
  - 下部: グレー "直近アクティビティ: 2 日前"
  - 右下: ボタン `[ 選択 ]`
- 6 枚を 3×2 グリッドで配置:
  1. 株式会社サンライズソーラー — 有効 — スコープ: 商談まで — 2 日前
  2. 関東ソーラー卸 株式会社 — 有効 — スコープ: 初回訪問まで — 5 日前
  3. グリーンエナジー商事 — 有効 — スコープ: アポ獲得まで — 1 日前
  4. 太平洋ソーラーパートナー — 有効 — スコープ: 商談まで — 12 日前
  5. 中部太陽光卸 — 停止中（淡グレー、ボタン非活性、注記 "関係停止のため業務操作不可"）
  6. 九州ソーラー商事 — 有効 — スコープ: 商談まで — 21 日前

### Section 3: 注記
- グレー "停止中の関係についても、過去の確定インセンティブの確認が可能です。"
```

## mobile.png プロンプト

```
Layout: 375x812 pixels.

Top bar (56px): "Solar SaaS" + アバター + "ログアウト" (ハンバーガーは不要)。

1. 見出し "提携卸業者を選択"
2. グレー案内文
3. 卸業者カード（縦に積む、各カード太枠、画面幅いっぱい）× 6 枚
   - 各カード: 卸業者名 / バッジ "有効" or "停止中" / スコープ表記 / 直近アクティビティ / `[ 選択 ]` ボタン
4. 最下部: グレー注記
```

## 設計意図メモ

- 多対多の核心。「今この瞬間、どの卸業者として動くか」を明示的に選ばせる
- 停止中の関係は薄表示 + 操作不可。ただし完全非表示にせず「過去の実績は閲覧可能」を案内
- 6 件程度の実データ風表示で、実運用での視認性を検証可能にする
- スマホでもカード型維持。各カードがタッチターゲットとして十分な面積を確保
