## 設計意図メモ

- 二次店管理者のみアクセス可。多対多関係の管理画面
- 卸業者ごとのデフォルトスコープを一覧化。卸業者-二次店関係 (F-009) の実体
- 関係停止は卸業者側操作。二次店側からは表示と切替操作のみ
- 新規提携は招待コード入力経由 (S-008)。明示的な導線
