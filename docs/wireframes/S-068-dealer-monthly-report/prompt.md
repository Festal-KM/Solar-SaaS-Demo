Main content:

### Section 1: パンくず + タイトル
- パンくず "ホーム > 月次報告 > 2026 年 5 月"
- 見出し "月次報告 - 2026 年 5 月"

- カード「成果 / 課題 / 翌月施策」:
  - ラベル "今月の成果（自社視点）" / テキストエリア (4 行)

  - ラベル "翌月の施策・要望" / テキストエリア (4 行)

### Section 5: 卸業者からのコメント（読み取り、参考）
- カード「卸業者本部からのコメント」:

### Section 6: フッター
- 右下: `[ 下書き保存 ]`(secondary) + `[ 提出 ]`(primary)
- グレー注記 "提出後でも、卸業者の確定前であれば再編集可能です。"

## 設計意図メモ

- 二次店の月次報告は「集計確認 + 3 段組コメント」の構成。F-049
- 暦月集計を明示。年度開始月の設定とは無関係（docs/01 §9.8）
- 仕入値非表示を注記で明確に。情報開示の境界
- 卸業者コメントは確定後に表示。それまでは「未到着」表示で待機状態を明示
