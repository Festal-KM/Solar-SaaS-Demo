## 設計意図メモ

- 卸業者本部はイベント当日 / 直後に「現場の状況」を確認する用途。報告タブを最初に出す
- スコープ上書きが入っているイベントは右上カードで明示。「契約条件と異なる」ことを警告に近い形で
- 二次店ごとの報告と自社報告を分離。共同開催では誰の責任か分かるように
- 警告バナーは「当日 + 開始未報告」のとき動的表示。普段は出ない
