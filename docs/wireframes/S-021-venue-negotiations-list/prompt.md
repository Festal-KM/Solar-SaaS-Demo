## 設計意図メモ

- 卸業者の「場所取り」業務の中核一覧。ステータスで「未連絡 → 調整中 → 確定」の進捗を可視化
- 次回アクション期日列で「今日明日やるべきこと」が即抽出可能
- 一括ステータス変更は複数選択時のみ活性（バルク操作）
