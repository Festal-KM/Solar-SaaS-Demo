## 設計意図メモ

- 招待詳細を読み取り専用カードで示し、ユーザーが「自分が誰として何のロールで参加するか」を確認できるようにする
- メールアドレスは変更不可。招待時に確定しているため改ざんを防ぐ
- 必須ロールの 2FA 強制は事前に注記し、後続 S-003 への遷移で驚かないようにする
