## 設計意図メモ

- F-055 の核。卸業者管理者が組織内の操作履歴を監査
- PII は監査ログ内でも常時マスク。RevealPii 操作自体もログに残るがログ表示時には PII はマスク
- 行展開で before/after の JSON 差分。具体的な変更内容を確認可能
- 90 日以前のアーカイブ表示は明示的にチェックボックスで切替（パフォーマンス配慮）
- 「追記専用」を注記で明示
