## 設計意図メモ

- 業務操作不可状態であることを最大限明示。サイドバーを出さないことで導線そのものを物理的に遮断
- ただし「過去の確定インセンティブ」だけは確認可能。F-009 の業務ルール（既存確定分は保持）に従う
- 関係を再開するには「相手側に依頼」が必要であることを明示。本人操作で解除できないことを伝える
- 新規卸業者との提携は別経路として招待コード入力リンクを提供
