## desktop.png プロンプト

```
Layout: 1440x900 pixels.

Main content (中央寄せ最大 900px):

### Section 1: パンくず + タイトル
- パンくず "ホーム > 設定 > 通知設定"
- 見出し "通知設定"

- カード「通知チャネル」:
  - 行 1: アプリ内通知 / `[ON]` (常時 ON、淡グレーで変更不可)

- カード「種別ごとの通知設定」:

    | 種別 | アプリ内 | メール | LINE |
    | 二次店からの希望提出 | `[ON]` | `[ON]` | `[OFF]`(淡) |
    | 開催体制決定 | `[ON]` | `[ON]` | `[OFF]`(淡) |
    | マエカク結果到着 | `[ON]` | `[OFF]` | `[OFF]`(淡) |
    | 商談ステータス変更 | `[ON]` | `[OFF]` | `[OFF]`(淡) |
    | 契約成立 | `[ON]` | `[ON]` | `[OFF]`(淡) |
    | キャンセル発生 | `[ON]` | `[ON]` | `[OFF]`(淡) |
    | 月次報告コメント | `[ON]` | `[OFF]` | `[OFF]`(淡) |
    | 月次確定 | `[ON]` | `[ON]` | `[OFF]`(淡) |

### Section 4: 連絡先カード
- カード「通知先メールアドレス」:
```

## 設計意図メモ

- アプリ内通知は常時 ON（運用上必須）
- LINE は Phase 2。UI 上は項目を出すが操作不可で「Phase 2 で対応予定」を明示
- 種別ごとに 3 チャネルの組み合わせを設定可能。きめ細かい制御
- 連絡先メールはプロフィール S-081 へリンク
