## 設計意図メモ

- 3 ステップウィザード（コード入力 → 会社情報 → 管理者アカウント）の起点。全体感を可視化
- 招待コード形式はサンプルで提示。マスキング表現で具体的な値漏えいを避ける
- エラー文言は無効/期限切れ/上限到達の 3 ケースを 1 文に集約。攻撃者に内訳を渡さない
