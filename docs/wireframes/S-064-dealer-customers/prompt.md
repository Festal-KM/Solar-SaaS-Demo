## 設計意図メモ

- 自社関与のみ表示（F-032）。他社二次店の登録顧客は不可視
- 仕入値カラム自体が API レベルで削除されているため、UI 設計上も存在しない
- マスク + 鍵アイコンは S-032 と同様だが、二次店側では権限により鍵活性条件が異なる
- フィルタ群は S-032 と同等。検索動作も同様
