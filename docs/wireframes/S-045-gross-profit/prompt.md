## 設計意図メモ

- F-042 / F-047 の核。実販売価格 − 仕入値 − 施工費 − その他 = 案件粗利、をフォームで明示
- インセンティブ対象粗利の選択肢を 3 つ（案件粗利 / 卸粗利 / 手動指定）。業務都合で切替可能
- 共同開催の按分は最重要 UI。デフォルト按分 + 下書き + 確定の 3 列で過程を見せる
- 履歴は監査用途。手動調整は誰がいつ何を変えたか追跡可能
