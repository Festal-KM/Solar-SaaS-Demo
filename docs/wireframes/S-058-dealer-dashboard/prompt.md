## 設計意図メモ

- 二次店本部は「現在の卸業者コンテキスト」で表示が切り替わる。ヘッダーのテナント切替を最上部で明示
- 当月 KPI は自社関与分のみ。F-051 / docs/01 §9 の情報開示制約
- 希望提出期限の近い候補は期限カウントダウン付き。提出忘れを予防
- 通知サマリは具体的な業務リンク付き。「マエカク結果到着」「開催体制決定」など二次店特有
- 仕入値・場所提供元契約条件は表示しない（カード内のデータ項目から物理的に除外）
