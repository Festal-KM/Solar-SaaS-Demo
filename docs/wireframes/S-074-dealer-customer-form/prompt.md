## 設計意図メモ

- S-057（自社現場）と同型。違いはチャネル選択（二次店は催事 / 飛び込み / テレコール / 紹介 / その他から選択）
- 催事チャネル時のみ獲得イベントが必須 (F-031 受入基準)
- 登録 → 卸業者のマエカクキューへ自動投入。注記で副作用を明示
- 重複検出ロジックは S-057 と同一
