## 設計意図メモ

- 2 ペイン構造で複数案件の連続確認を効率化
- 施工費用フィールドは粗利再計算のトリガー。注記で副作用を明示
- 施工予定 7 日前以内は通知バッジで強調（ワイヤーで 1 件サンプル化）
- 添付は現地調査 PDF / 施工計画 / 進捗写真など。施工管理の証跡集約
