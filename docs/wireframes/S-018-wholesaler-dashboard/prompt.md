## 設計意図メモ

- アラート最上段はホーム画面の心臓。卸業者本部の「今日やるべきこと」リスト。各行は具体画面へ直接遷移
- KPI は当月集計を即可視化。前月比で異常検知できるように
- カレンダー + 通知の 2 カラムは「予定」と「履歴」の対比。卸業者は両方を毎日確認する
- スパークラインは BI への玄関口。詳細は S-051 へ
- 空状態は卸業者向けのチュートリアル化。マスタ整備→候補登録の動線を明示
