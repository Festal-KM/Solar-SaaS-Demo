## 設計意図メモ

- イベント実施状況を一覧で監視。シフト充足度と報告状況を列で並べ、ステータスの代理指標として表示
- 当日イベントは highlight（太枠 + 強調バッジ）。卸業者本部の「今日の現場」を即可視化
- カレンダービュー切替を提供。月単位での負荷バランスを把握する用途
- 中止イベントも一覧には残す（淡グレー）。完全削除しない（履歴保持）
