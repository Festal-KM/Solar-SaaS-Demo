## 設計意図メモ

- 配属確定後のイベントを月別管理。自社が担当 or 補助する範囲のみ表示（他二次店の配属は不可視）
- スコープ列で「アポ獲得まで / 初回訪問まで / 商談まで」を明示。イベント単位上書きはタグで強調
- 報告未提出のイベントは warning ハイライト。終了済みでも未提出は明確に
- 自社の役割「主担当 / 補助」を表示。共同開催時の責任範囲を可視化
