## 設計意図メモ

- スコープによって編集権限が変わる。アポ獲得までスコープのときは全フィールド灰色化（このワイヤーは「商談まで」スコープで例示）
- 仕入値は非表示。粗利見込みは「自社経費ベース」のラベル付き
- 契約ステータス昇格は重い操作。注記で「卸業者に通知される」「契約金額必須」を明示
- ステッパーは商談-クロージング-契約までの 5 段階。契約は卸業者責任
