## 設計意図メモ

- 二次店共有設定はデフォルト全選択 + 個別チェック外し。F-019 の「対象二次店設定」を最も使いやすく
- 停止中の二次店は淡グレーでチェック不可（自動）。卸業者が誤って公開しないように
- 卸業者内部メモは「二次店には非表示」を見出しに明示。情報開示の境界を意識させる
- 回答期限の警告（実施日より後）はワイヤーで注記として残す
