## 設計意図メモ

- オフライン記録であることを最上部で明示。「決済処理は別系統」の宣言で運営者の誤解を防ぐ
- ステータス「督促」は視覚的に強調。10 日経過などのメモを併記して対応優先度を可視化
- 月切替は最重要操作のため最上段配置
- スマホでもカード型に切替えるが、金額とステータスを最大限大きく表示
