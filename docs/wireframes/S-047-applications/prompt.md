## 設計意図メモ

- 申請は契約に複数紐づく（国補助金 / 系統連系 / 自治体補助金 / 売電契約 等）。1 契約 - 多申請の構造
- 申請期限 14 日前以内は warning。F-045 の通知ルールに直結
- ステータスは「未着手 → 書類準備中 → 申請済 → 審査中 → 承認/却下」のフロー
- 添付ファイルは申請書類本体や添付資料。後追い監査で参照
